## Description
A Flutter application that allows users to search, browse, and explore free books using the Google Books API. Built with a focus on scalability and maintainability, the app follows the MVVM (Model-View-ViewModel) design pattern.

## Features
- Built using the Google Books API for book data.
- View detailed information about books, including title, author, rating, and cover image
- Allows to preview free books from Google Books.
- Search for books by title.
- Responsive and user-friendly UI.
